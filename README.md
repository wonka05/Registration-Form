# Registration Form

```text
A simple user registration system built with PHP, SQLite and HTML/CSS. It allows users to submit their details via a registration form and view registered users on a separate page.
```

## Features:

```text
- User registration with form validation
- Stores user data in a local SQLite database
- View list of registered users
- Easy to set up and customize
```

## Tech Stack:

```text
PHP for backend logic
HTML & CSS for frontend form UI
SQLite for database     
```

## Project Structure: 

```text
/
├── README.md           
├── index.php             # Registration form & form handling
├── view_users.php        # Page to display registered users
├── data.sqlite           # SQLite database storing registrations

```

## Setup & Installation: 
```text
1. Clone the Repo:
      git clone https://github.com/manasag1605/Registration-Form.git

2. Move files to your web server directory
      For XAMPP:
        C:\xampp\htdocs\Registration-Form

3. Start your server
      Start Apache 

4. Open in browser:
      http://localhost/Registration-Form/index.php

```
